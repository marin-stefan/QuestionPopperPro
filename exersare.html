Exersează întrebări – JavaScript

Domeniu: JavaScript  |  Dificultate: Mediu  |  Frecvență: Ridicată

Întrebare: Ce este „hoisting” în JavaScript și cum afectează variabilele și funcțiile?

Răspuns: „Hoisting” este un comportament specific JavaScript prin care declarațiile de variabile (cu var) și funcțiile declarate sunt „mutate” la începutul contextului lor de execuție, înainte ca codul să fie rulat. Astfel, poți utiliza o funcție sau o variabilă înainte să fie definită în cod (cu unele limitări pentru let și const care nu sunt „hoisted” complet).
